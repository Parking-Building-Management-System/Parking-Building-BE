Step 2: Swagger - Presign Upload

Endpoint:
POST /manager/storage/presign-upload

Swagger tag:
Manager Storage

Controller:
src/main/java/com/smartpark/swp391/modules/manager/controller/ManagerStorageController.java

Method:
presignUpload

Request DTO:
PresignUploadRequest

Request:
{
  "fileName": "demo-parking-1.png",
  "contentType": "image/png",
  "folder": "floor-maps"
}

Expected response:
{
  "objectKey": "tenants/<tenantId>/floor-maps/<uuid>-demo-parking-1.png",
  "uploadUrl": "http://localhost:9000/smartpark/...",
  "method": "PUT",
  "headers": {
    "Content-Type": "image/png"
  },
  "expiresInSeconds": 900,
  "publicUrl": null
}

Fields can copy:
* uploadUrl
* objectKey
* expiresInSeconds

Validation:
* Auth/role: PARKING_MANAGER. DEV pass vi co effective manager authorization.
* Tenant: lay tu token/TenantContext.
* folder: optional, max 120, chi cho phep A-Z a-z 0-9 / _ -, khong duoc bat dau bang /, khong duoc co ..
* contentType allowed: image/png, image/jpeg, image/webp.
* extension phai khop contentType: png -> .png, jpeg -> .jpg/.jpeg, webp -> .webp.
* fileName: max 255, strip path segment, thay ky tu la bang -, khong cho blank, "." hoac "..".
* objectKey format: tenants/{tenantId}/{folder}/{uuid}-{sanitizedFileName}
* Max file size: presign request hien tai chua validate size. Gioi han thuc te nam o client/UI hoac MinIO.

PHAN 6: TROUBLESHOOTING 400

Symptom: 400 SignatureDoesNotMatch / BadRequest
Likely cause: PUT Content-Type khac voi contentType luc xin presign.
Fix: Neu presign la image/png thi PUT phai co header -H "Content-Type: image/png" y het.

Symptom: 400 SignatureDoesNotMatch / BadRequest
Likely cause: Dung POST hoac multipart/form-data.
Fix: Dung PUT raw binary: curl -X PUT --data-binary "@/path/file.png". Khong dung form-data.

Symptom: 403 AccessDenied khi goi backend presign
Likely cause: Token sai role hoac het han.
Fix: Login DEV, paste access token vao Swagger Authorize hoac Bruno managerToken.

Symptom: 403 AccessDenied khi PUT len MinIO
Likely cause: UploadUrl het han hoac bi sua query string.
Fix: Xin presign upload moi, copy nguyen uploadUrl.

Symptom: 403 AccessDenied khi curl -I downloadUrl
Likely cause: Download URL duoc ky cho GET, nhung curl -I dung HEAD.
Fix: Dung browser hoac curl GET: curl -o /tmp/smartpark-presign-demo.png "$DOWNLOAD_URL".

Symptom: 404 NoSuchBucket
Likely cause: APP_MINIO_BUCKET sai hoac bucket chua duoc tao.
Fix: Kiem tra MinIO running va app config bucket. App co initializer tao bucket khi credentials hop le.

Symptom: 404 NoSuchKey
Likely cause: Chua PUT file thanh cong, objectKey sai, hoac copy objectKey cua lan presign khac.
Fix: PUT lai dung uploadUrl/objectKey vua xin.

Symptom: Unsupported API call for method GET at //smartpark
Likely cause: Endpoint MinIO co trailing slash va client cu sinh URL double slash.
Fix: Code da normalize endpoint. Van nen de APP_MINIO_ENDPOINT=http://localhost:9000 de de doc.

Symptom: Browser PUT truc tiep bi CORS
Likely cause: MinIO CORS chua allow origin/header/method PUT.
Fix: Demo bang curl/Bruno truoc. Neu demo browser, cau hinh MinIO CORS allow PUT va Content-Type.

Symptom: Backend bao unsupported content type
Likely cause: contentType khong thuoc image/png, image/jpeg, image/webp.
Fix: Dung file PNG/JPEG/WebP that va khai bao dung Content-Type.

Symptom: Backend bao extension/content type mismatch
Likely cause: Vi du file .png nhung contentType image/webp.
Fix: Dung demo-parking-1.png voi image/png.

Symptom: Presign URL expired
Likely cause: De qua 900 giay.
Fix: Xin presign lai.

Symptom: APP_MINIO_ENDPOINT trailing slash
Likely cause: Env de http://localhost:9000/.
Fix: Code da trim trailing slash; van co the sua env ve http://localhost:9000 de tranh nham lan khi debug.
